🛰️ Ferramenta de Comunicação em Rede
📑 Descrição
Este projeto tem como objetivo o desenvolvimento de uma ferramenta para comunicação em rede, utilizando os protocolos e recursos de redes baseados em TCP/IP. A aplicação permite que dois ou mais usuários se comuniquem entre si através de uma rede local ou internet, transmitindo dados por meio de sockets.

🔧 Funcionalidades
Comunicação ponto a ponto ou em grupo.

Troca de mensagens em tempo real utilizando TCP/IP.

Suporte a transmissão de dados via sockets.

Extensível para funcionalidades adicionais, como:

Envio de arquivos.

Emojis e componentes gráficos.

Comunicação por áudio, vídeo ou webcam.

Comunicação multicast.

Integração com e-mail e notificações.
